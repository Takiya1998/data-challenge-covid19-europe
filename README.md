# COVID-19 EU Data Challenge

## Contexte
Ce dépôt présente un projet académique réalisé dans le cadre d’un data challenge consacré à l’analyse des données COVID-19 en Union européenne.

## Objectif
L’objectif était de nettoyer, harmoniser et explorer des données épidémiologiques afin de produire des visualisations interactives permettant de suivre l’évolution des cas confirmés en Europe.

## Travaux réalisés
- uniformisation des variables entre plusieurs fichiers CSV
- détection et correction de valeurs aberrantes
- gestion des valeurs manquantes
- création d’un graphique interactif temporel
- création d’une carte choroplèthe animée
- développement d’un dashboard interactif avec Dash

## Technologies utilisées
- Python
- pandas
- numpy
- plotly / plotly express
- Dash
- glob

## Visualisations produites
- courbes temporelles interactives
- filtre par année
- légende interactive et infobulles
- carte de l’Union européenne animée dans le temps
- dashboard avec carte, barplot, camembert et tableau des pays les plus touchés

## Cadre académique
Projet réalisé en groupe dans le cadre d’un data challenge universitaire.
